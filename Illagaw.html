About Illagaw

Illagaw is a location-based game that showcases the rich food, arts, and heritage of Iloilo. The game allows users to explore the city's landmarks while engaging in interactive tasks and challenges designed to promote Iloilo's unique culture.

The app features a trained object detection model, enabling users to interact with real-world objects as part of the gameplay.

Key Features

  • Highlights Iloilo's food, arts, and heritage.
  • Interactive location-based challenges.
  • Real-time object detection technology.
  • Engaging and educational gameplay.
